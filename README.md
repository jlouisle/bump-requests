# Bump Requests

Install node and npm
`brew install node`

Install the Angular CLI globally.
`npm install -g  @angular/cli`

Serve Angular
`cd bump-requests`
`ng serve -o`

### Generate [blueprint]
`ng generate [blueprint] [name]`

__[blueprint]__:
  * component
  * module
  * class
  * directive
  * enum
  * guard
  * interface
  * pipe
  * service
